# 首页优化说明

## 优化概览

首页经过全面重新设计，采用现代化的界面风格和更好的用户体验。

## 主要改进

### 1. 视觉设计升级

#### 渐变色主题

- **按钮**: 带渐变背景和悬停阴影效果
- **卡片**: 半透明毛玻璃效果 (backdrop-filter)

#### 动画效果
- **Logo**: 脉动缩放动画 (pulse)
- **背景**: 浮动扑克牌符号装饰 (♠♥♣♦)
- **卡片**: 悬停时上移和发光效果
- **图标**: 悬停时旋转和放大

### 3. 新增功能

#### 快速导航
所有主要功能页面都可以从首页一键访问：
- **创建游戏** → 打开游戏创建对话框
- **数据分析** → 跳转到 `/analytics`
- **玩家统计** → 跳转到 `/stats`
- **游戏模拟** → 跳转到 `/simulation`

#### 实时统计
首页自动加载并显示系统统计数据：
- 调用 `GET /api/analytics/overview?days=30`
- 显示总游戏数、已完成、总底池、平均时长
- 数据实时更新

### 4. 响应式设计

#### 移动端适配
- 小屏幕下卡片自动换行
- 标题字体缩小 (56px → 36px)
- 按钮变为纵向排列
- Logo 缩小 (80px → 60px)

#### 断点设置
- `xs`: 手机竖屏 (< 768px)
- `sm`: 手机横屏/小平板 (≥ 768px)
- `md`: 平板/小电脑 (≥ 992px)
- `lg`: 桌面电脑 (≥ 1200px)

### 5. 交互体验优化

#### 悬停效果
- **卡片**: 上移 8px，发光边框
- **按钮**: 上移 2px，增强阴影
- **图标**: 放大 1.2倍，旋转 5度

#### 平滑过渡
所有动画都使用 `transition: all 0.3s ease`

#### 视觉反馈
- 悬停时显示 "点击进入 →"
- 按钮按下时有缩放效果
- 加载状态有 loading 图标

## 技术实现

### 关键技术点

#### 1. CSS渐变
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
```

#### 2. 毛玻璃效果
```css
background: rgba(255, 255, 255, 0.05);
backdrop-filter: blur(10px);
```

#### 3. 动画关键帧
```css
@keyframes float {
  0%, 100% { transform: translateY(0) rotate(0deg); }
  50% { transform: translateY(-30px) rotate(10deg); }
}
```

#### 4. 数据格式化
```javascript
// 大数字格式化
formatLargeNumber(num) {
  if (num >= 1000000) return (num / 1000000).toFixed(1) + 'M'
  if (num >= 1000) return (num / 1000).toFixed(1) + 'K'
  return num.toLocaleString()
}

// 时长格式化
formatDuration(seconds) {
  const mins = Math.floor(seconds / 60)
  const secs = Math.floor(seconds % 60)
  return `${mins}:${secs.toString().padStart(2, '0')}`
}
```

## 部署步骤

### 本地开发
```bash
cd frontend
npm install
npm run dev
```

### 生产构建
```bash
cd frontend
npm run build
```

### Docker 部署
```powershell
# 使用提供的脚本
.\restart-local.ps1
```

或手动执行：
```bash
docker compose down
docker compose build --no-cache frontend
docker compose up -d
```
